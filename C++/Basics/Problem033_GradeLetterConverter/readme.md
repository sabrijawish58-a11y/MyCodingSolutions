# 🅰️ Grade Letter Converter

## 🧩 Problem Statement
Write a C++ program that asks the user to enter a grade between 0 and 100, validates the input, and then converts the numeric grade to a letter grade based on the following scale:

- 90–100 → A  
- 80–89 → B  
- 70–79 → C  
- 60–69 → D  
- 50–59 → E  
- Below 50 → F
